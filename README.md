# 🇷🇺 Росбраузер (RosBrowser) — Полная Переработка

Добро пожаловать в проект **«Росбраузер» (RosBrowser)** — флагманский, суверенный, производительный и визуально безупречный веб-браузер на базе современного движка WebView (Chromium) с глубокой интеграцией отечественных сервисов и революционным премиум-дизайном в концепции стеклянного морфизма (Glassmorphism).

Этот репозиторий представляет собой полностью переработанную версию браузера, адаптированную для безупречной работы на устройствах Android (AOSP, API 26-35) с поддержкой автоматической сборки и публикации (CI/CD).

---

## 💎 1. Дизайн: Премиум-интерфейс в Летней Стилистике

### 1.1 Живой летний пейзаж (New Tab Page)
Стартовая страница содержит процедурно генерируемую интерактивную графику:
*   **Динамический небосвод:** градиенты меняются в реальном времени в зависимости от времени суток на устройстве (утро / день / вечер / ночь).
*   **Параллакс облаков:** 3 независимых слоя облаков с процедурным сдвигом, создающие иллюзию глубины.
*   **Патриотическая палитра (Триколор):** Белые облака, синий небосвод и поле покачивающихся алых маков образуют живой Российский Флаг.
*   **Голографический флаг:** Водяной знак флага РФ с переливом (альфа-канал 10%), смещающийся при движении гироскопа.
*   **Интерактивный пруд:** Анимированная водная рябь с плавающими кувшинками, бабочками и кувыркающимися божьими коровками.

### 1.2 Инновационная адресная строка (Omnibox)
*   **Парящий стеклянный дизайн:** Нижнее расположение строки, закругление углов 20dp, размытие заднего фона 30dp.
*   **Интерактивность:** Пружинные плейсхолдеры и анимации при фокусе, цветные градиентные тени, анимированные иконки микрофона, замка безопасности и QR-сканера.
*   **Умный поиск:** Асинхронные подсказки с таймаутом 3с и автоисправлением русской/английской раскладки.

### 1.3 Стеклянное Табло быстрого доступа
*   **«Морские камушки»:** Закругленные плитки (20dp) с песочно-янтарным градиентом и тонким бликом.
*   **Анимации:** Мягкое вдавливание при клике, перенос элементов (Drag-and-Drop) с магнитным притяжением и удаление с эффектом рассыпания на пиксели.
*   **Пульсирующая плитка «+»** для добавления любимых веб-сайтов.

## 🛡️ 2. Банановая Защита 🍌 и Админ-Панель

В разделе «Настройки» («Банановая Защита») и на встроенной странице `chrome-native://admin` пользователям доступен полный спектр инструментов кибербезопасности:
*   **Анти-трекинг (Банановый След) 🐾:** Блокировка следящих веб-маяков, пикселей и скриптов метрик.
*   **Блокировка рекламы (Банановый Щит) 🛡️:** Полное вырезание рекламных фреймов на базе списков EasyList Russia и RuAdList.
*   **Контроль соединений (Банановый Патруль) 🚨:** Автоматическое предупреждение пользователя о посещении небезопасных (HTTP) сайтов, предотвращающее перехват личных данных.
*   **Полный дашборд:** Графики заблокированных трекеров, объём сэкономленного трафика и мониторинг системных ресурсов.

## 🛠️ 4. Локальная сборка через Docker

Для сборки стабильного дистрибутива APK в изолированном контейнере без необходимости установки Android Studio на хост-компьютере, выполните следующие команды:

```bash
# 1. Сборка Docker-образа компилятора
docker build -t rosbrowser-builder .

# 2. Извлечение готового APK файла из контейнера
docker create --name ros_container rosbrowser-builder
docker cp ros_container:/app/rosbrowser-debug.apk ./rosbrowser-debug.apk
docker rm ros_container

echo "Сборка успешна! Файл ./rosbrowser-debug.apk готов к тестированию."
```

## 🚀 5. Автоматическая сборка (GitHub Actions)

В проекте настроен полноценный конвейер непрерывной интеграции (CI/CD):
1.  **Триггеры запуска:** Автоматический запуск сборки при любом `push` в ветку `main` или добавлении релизного тега вида `v*`.
2.  **Сценарии (`build_rosbrowser.yml` и `android.yml`):**
    *   Клонируют репозиторий.
    *   Разворачивают среду **Java JDK 17 (Zulu)**.
    *   Кэшируют зависимости Gradle для молниеносной повторной сборки.
    *   Проводят декодирование ключей подписи.
    *   Компилируют проект и публикуют подписанный дебаг-ключом файл `app-debug.apk` во вкладку **Artifacts** рабочего процесса.

## 🔧 6. Набор патчей к Chromium (Chromium Forks)

Поскольку Росбраузер базируется на оптимизированном форке ядра Chromium, в папке `/chromium_patches/` подготовлен полный пакет патчей с комментариями ведущих разработчиков:
*   `patch_gost_crypto.diff` — Интеграция шифрования ГОСТ Р 34.10-2012 / libgost в библиотеку BoringSSL.
*   `patch_rkn_blocking.diff` — Механизм низкоуровневой блокировки РКН на этапе URLLoader в Chromium Network Service.
*   `patch_turbo_mode.diff` — Алгоритмы транскодирования тяжелых веб-ресурсов в WebP / Brotli через российские прокси.

## 📐 7. Стабильность и архитектура
Приложение спроектировано по архитектурному паттерну **MVVM + Clean Architecture**:
*   **State Management:** Полностью реактивный поток данных на базе `MutableStateFlow` и `collectAsStateWithLifecycle` предотвращает утечки памяти.
*   **Безопасность:** Отсутствие падений (Zero-Crash Policy) благодаря глобальным обработчикам `try-catch` во всех фоновых процессах и операциях ввода-вывода.
*   **Edge-to-Edge:** Использование Material Design 3 с поддержкой системных отступов (`WindowInsets`).
